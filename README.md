# ☁️ Cloud Gallery

A React-based interactive gallery showcasing manga titles in a stylized “cloud” UI concept. Built to demonstrate component design, modal systems, responsive layout, and image optimization techniques.

---

## 🚀 Live Demo
https://cloud-gallery-eight.vercel.app/

---

## 🛠️ Tech Stack
- React (Vite)
- SCSS Modules
- JavaScript (ES6+)
- CSS Flexbox / Responsive Design

---

## ✨ Features
- Interactive cloud-style landing UI
- Modal system with nested detail views
- Responsive design (desktop + mobile)
- Image preloading for smooth UX
- Component-based architecture
